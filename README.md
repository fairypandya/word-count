Word Count Tool

A simple and interactive Word Count web application that counts words, characters, and other text statistics in real time as the user types.

About the Project

This project was built to practice JavaScript DOM manipulation and create a small but useful utility.
It provides instant feedback on text input, making it helpful for writing, editing, and basic text analysis.

Features

Real-time word count

Character count (with and without spaces)

Live updates as the user types

Clean and minimal UI

Works entirely in the browser (no backend)

Tech Stack

HTML – Structure

CSS – Styling and layout

JavaScript – Logic and interactivity

Live Demo:https://fairypandya.github.io/wordcn/

What I Learned

Handling user input with JavaScript

Updating the DOM dynamically

Writing clean and reusable functions

Improving UI responsiveness with minimal code
